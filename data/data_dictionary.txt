Data Dictionary for RP_SE282_RegrCoeffs and RP_SE282_SegProfileCoeffs

When creating a StemProfileModel() the following parameter values are allowed.
An asterik (*) denotes the default value if none is supplied.

region:
	southwide
	coastal plain
	upper coastal plain
	piedmont
	(*) deep south

spp: species
	all pines combined
	shortleaf pine
	slash pine
	longleaf pine
	(*) loblolly pine
	white pine
	virginia pine
	cherry
	all hardwoods combined

bark:
	0 = outside bark
	(*) 1 = inside bark

dbh: diameter at breast height (4.5 ft above groundline)
	any integer or decimal value is permitted

height: total tree height to the tip in feet
	any integer or decimal value is permitted
